Absolute Media Inc System Upgrade

An Updated Database System for the employees of Absolute Media Inc Printing Company.


My Role: UI Designer, Front-End Developer
Project: University
Timeline: 9 mo; September 2021 - May 2022
Tools Used: Figma, Figjam
Languages Used: HTML, CSS, ReactJS

Overview

Prompt

The Absolute Media Inc printing company wants to update their company database system for employees to have a smooth experience when working and getting forms out. The old system ran on Windows 7 and needs serious modern capabilities installed all while keeping a simple outlook visually.

My responsibility: create login and dashboard pages that satisfy the overall feel for the end product.

Design Decision

Q. Which is most efficient, update from old system or create a brand new one?

A. Create a brand new one.

The decision to scrap the old system to create a new one came light of the team not wanting to have to read through old code and design methods. The system was many moons behind the current trends to where the team decided that in order for the system to be up to date, it would be better and faster to create anew rather than revamp and redesigning from the old base system.

Reflection

Takeaways

This part of the project has focused on research and identified pain points and needs. Knowing what it is the users wants, there is generally a solution for it. Research into modern examples and user personas enabled me to understand the target users more in order to figure out the best, most efficient design solution.

The client, after showing them the prototypes, generally liked how simple and easy to follow the elements are.
